# Implementing Base64 with python

## Installation

```bash
git clone git@github.com:mohpezeshki/base64.git
```

## Run

### Decode
```bash
python base64.py encode "PLAIN\_TEXT"
```

### Decode

```bash
python base64.py decode "CIPTHER\_TEXT"
```
